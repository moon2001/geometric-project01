What is DOM and what is the purpose of DOM?

DOM stands for document object model.It is a programming interface for web documents. It defines logical structure of document.

The purpose of DOM is speeding up the web page and allow to create,change, remove element from a website


How will you select HTML elements using DOM? Name the DOM methods.

There are most popular 5 ways/methods to select HTML elements in a DOM using selectors. Those are:

  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

What is event bubble?

When an element receives a event which is bubbling up in the process of receive until it reached to the root element that is called event bubble in Dom


What is a callback function and why will you use it?

callback function: When a function passed into another function as an argument then provoked the inside or outer function to complete the needed action is called callback function

A callback function is used to execute a function sequentially and appropriately
